# Parameterization

Hamilton and mutualism simulation data use K = 0.5 and b = 0.4 (b fixed); c
varies from 0 to b.

- Hamilton/mutualism PD: T = K + b, R = K + b - c, P = K, S = K - c
- Hamilton/mutualism SD: T = K + b, R = K + b - c/2, P = K, S = K + b - c

Result folders are named by dilemma type:

- **0** — control: T = P, R = S; partner behavior does not affect focal payoffs; interactive cooperation is not in the game
- **1** — prisoner's dilemma: T = K + b, R = K + b − c, P = K, S = K − c
- **2** — snowdrift: T = K + b, R = K + b − c/2, P = K, S = K + b − c
